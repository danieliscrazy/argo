# Argo
This is Argo, the mainboard powering the Talos starter program, a (hopefully) upcoming Hack Club YSWS program. It has an onboard DRV8833 that drives 2 DC motors, headers for a Raspberry Pi Pico, as well as a JST power connector, a power switch, and a voltage regulator to lower the 4xAA battery pack to an appropriate voltage for the Pi Pico.
